geocommit

Use

Only WiFi is required. No GPS needed.

Git

$ easy_install geocommit
$ cd myproject
$ git geo setup
$ git commit
$ git geo push

Mercurial

$ hg cl http://bitbucket.org/segv/hg-geo
$ echo "[extensions] geo=" >> ~/.hgrc
$ hg commit

Integrate

Add geocommit support to github and bitbucket. Install the Google Chrome extension or the Greasemonkey User Script for Firefox.

Hooks

To visualize your repository's commits on a map, use our Post-Receive Url service.

github

Add http://hook.geocommit.com/api as a Post-Receive Url.

bitbucket

Add http://hook.geocommit.com/api as a POST service.

The Geocommit Format 1.0

The geocommit data consists of key/value pairs. Keys are lowercase alphanumeric identifier including _ and -. Values are numbers or strings not containing linebreaks, commas and colons.

Short Format

geocommit(1.0): <key> <value>, ..., <key> <value>;

Long Format

The long format is, terminated by an empty line: 
geocommit (1.0)
<key>: <value>
...
<key>: <value>

Defined keys

Key Required Description
long ylongitude value of a coordinate in WGS84
lat ylatitude value of a coordinate in WGS84
src ydata provider used to generate the geodata
alt naltitude in metres
speednspeed in m/s
dir ndirection of travel
hacc nhorizontal accuracy of long/lat values in metres
vacc nvertical accuracy of altitude value in metres
